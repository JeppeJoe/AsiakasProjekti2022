## Vaatimusmääritelmä

Tehdään:

Nettisivu, jossa on aloitussivu, jonka kautta pystyy navigoimaan palkkien kautta tarinoihin. Nettisivu tehdään todennäköisesti käyttäen wordpressiä.
#
